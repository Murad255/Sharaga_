Контроль ориентации

Описание

Постоянная ориентация

Ориентация TCP во время движения остается постоянной.

Ориентация начальной точки сохраняется. Запрограммированная ориентация целевой точки не учитывается.

Стандартный

Ориентация TCP во время движения постоянно меняется. В целевой точке у TCP имеется запрограммированная ориентация.

Без ориентации

Данная опция доступна только для сегментов Spline. (Не для блока Spline или для отдельных перемещений Spline).

Она применяется, если для какой-либо точки не требуется определенной ориентации.

Без ориентации

Опция Без ориентации используется, если для точки не требуется определенной ориентации. Если выбрана эта функция, заученная или запрограммированная ориентация этой точки игнорируется. Вместо этого система управления роботом рассчитывает оптимальную ориентацию для данной точки, исходя из ориентаций окружающих точек.

Особенности опции Без ориентации:

■

При виде выполнения программы MSTEP и ISTEP робот останавливается на ориентациях, рассчитанных системой управления роботом.

■

При выборе записи по пункту Без ориентации робот принимает ориентацию, рассчитанную системой управления роботом.

Для следующих сегментов выбор Без ориентации запрещен:

■

первый сегмент в данном блоке Spline

■

последний сегмент в данном блоке Spline

■

сегменты SCIRC Контроль ориентации круговой траектории = относительно траектории

■

сегменты, за которыми следует сегмент SCIRC Контроль ориентации круговой траектории = относительно траектории

■

сегменты, за которыми следует сегмент Контроль ориентации = Постоянная ориентация

■

Если несколько сегментов с одинаковыми целевыми точками в декартовых координатах следуют друг за другом, то опция Без ориентации для первого и последнего сегментов запрещена.

SCIRC

Для движений SCIRC имеется такой же выбор возможностей контроля ориентации, что и для движений SLIN. Дополнительно для движений SCIRC можно определить, должен ли контроль ориентации быть соотнесен с пространственной ориентацией или с траекторией.

Для следующих движений опция По отношению к траектории запрещена:

■

сегменты SCIRC, для которых действует контроль ориентации Без ориентации;

■

движения SCIRC, которым предшествует сегмент Spline, для которого действует контроль ориентации Без ориентации.

Ориентация вспомогательной точки:

в случае движений SCIRC с контролем ориентации Стандарт система управления роботом учитывает запрограммированную ориентацию вспомогательной точки, однако лишь условно.

Начальная ориентация переходит в целевую ориентацию по пути, который включает в себя ориентацию вспомогательной точки. Т.е. ориентация вспомогательной точки принимается по пути, но не обязательно во вспомогательной точке.
