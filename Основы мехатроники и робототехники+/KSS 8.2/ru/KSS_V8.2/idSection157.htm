5.9.2 Индикация протокола техобслуживания

Описание

Запротоколированные работы по техобслуживанию можно просмотреть в специальном обзоре.

При обновлении системного программного обеспечения KUKA (например, с версии KSS 8.2.3 на версию KSS 8.2.4.) данный обзор сохраняется.

При выполнении архивирования всегда одновременно архивируются запротоколированные работы по техобслуживанию. При восстановлении данных, когда в системе управления роботом уже были запротоколированы другие работы по техобслуживанию, они не переписываются, а дополняются восстановленными протоколами.

Порядок действий

1.

В главном меню выбрать пункт Ввод в эксплуатацию > Сервисное обслуживание > Руководство по техобслуживанию. Открывается окно Руководство по техобслуживанию.

2.

Выбрать вкладку Обзор техобслуживания.
