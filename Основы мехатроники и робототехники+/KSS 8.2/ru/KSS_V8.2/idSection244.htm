8.3.4.11 Встроенный формуляр триггера Spline, тип «Установка импульсного выхода»

Рис. 8-20: Встроенный формуляр триггера Spline, тип Установка импульсного выхода

Поз.

Описание

1

Если команду необходимо сместить в пространстве, здесь необходимо указать требуемое удаление от начальной или целевой точки. Если пространственное смещение не требуется, ввести значение 0:

■

положительное значение: смещает команду в направлении конца движения;

■

отрицательное значение: смещает команду в направлении начала движения.

Только для группы пользователей «Эксперты»: Перекл. пути позволяет задать в этом поле переменную, постоянную или функцию. Для этих функций имеются определенные ограничения.

(>>> 8.3.4.14 "Ограничения для функций в триггере Spline")

2

С помощью Перекл. OnStart можно установить или удалить параметр ONSTART:

■

без ONSTART: значение PATH относится к целевой точке;

■

с ONSTART: значение PATH относится к начальной точке.

3

Если команду необходимо сместить по времени (относительно значения в поз. 1), здесь необходимо указать требуемую продолжительность времени. Если временное смещение не требуется, ввести значение 0.

■

положительное значение: смещает команду в направлении конца движения. Максимум: 1 000 мс;

■

отрицательное значение: смещает команду в направлении начала движения.

Только для группы пользователей «Эксперты»: Перекл. задержки позволяет задать в этом поле переменную, постоянную или функцию. Для этих функций имеются определенные ограничения.

(>>> 8.3.4.14 "Ограничения для функций в триггере Spline")

4

Номер выхода

■

1 … 4096

5

Состояние, в которое переключается выход

■

TRUE: уровень «High»

■

FALSE: уровень «Low»

6

Длина импульса

■

0,10 – 3,00 с
