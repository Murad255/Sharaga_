8.3.4.9 Программирование триггера в блоке Spline

Условие

■

Программа выбрана.

■

Выбран режим работы T1.

■

Формуляр блока Spline открыт.

Порядок действий

1.

Установить курсор в строку в блоке Spline, после которой должен быть вставлен триггер.

2.

Выбрать последовательность меню Команды > Логика > Триггер Spline.

3.

По умолчанию на экране показан встроенный формуляр Установка выхода. С помощью невидимой кнопки Перекл. типа можно вывести на экран другой формуляр.

4.

Во встроенном формуляре установить параметры.

5.

Нажать кнопку Команда OK.

Описание

Показываемый на экране встроенный формуляр зависит от типа, выбранного с помощью кнопки Перекл. типа.

Тип встроенного формуляра

Описание

Установка выхода

Этот триггер устанавливает выход.

(>>> 8.3.4.10 "Встроенный формуляр триггера Spline, тип «Установка выхода»")

Установка импульсного выхода

Этот триггер устанавливает импульс определенной длины. (>>> 8.3.4.11 "Встроенный формуляр триггера Spline, тип «Установка импульсного выхода»")

Триггер назначения

Этот триггер устанавливает значение для переменной. Доступно только для группы пользователей «Эксперты».

(>>> 8.3.4.12 "Встроенный формуляр триггера Spline, тип «Триггер назначения»")

Триггер вызова функции

Этот триггер вызывает подпрограмму. Доступно только для группы пользователей «Эксперты».

(>>> 8.3.4.13 "Встроенный формуляр триггера Spline, тип «Триггер вызова функции»")

Подробная информация о триггерах, смещении точки переключения и пределах смещения приведена в инструкции по эксплуатации и программированию для системных интеграторов.
