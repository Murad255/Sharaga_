2.4 Накопители USB KUKA

Для системы управления роботом KR C4 существуют следующие накопители USB KUKA:

■

Накопитель USB KUKA 2.0 NB 4ГБ

без возможности загрузки

№ арт. 00-197-266

■

Накопитель USB KUKA 2.0 Recovery 4ГБ

с возможностью загрузки

Компоненты изделия KUKA.RecoveryUSB 1.0, № арт. 00-198-642

Если для операции требуется использование определенного накопителя, это указано в описании этой операции.

Рис. 2-2: Накопитель USB KUKA 2.0 NB 4ГБ, без возможности загрузки (№ арт. 00-197-266)

Рис. 2-3: Накопитель USB KUKA 2.0 Recovery 4ГБ, с возможностью загрузки (№ арт. 00-198-642)
