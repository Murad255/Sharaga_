6.1.3 Создание новой программы

Условие

■

На дисплее появляется навигатор.

Порядок действий

1.

Выделить папку в структуре папок, в которой следует создать новую программу, например, папку Program. (Новые программы можно создать не во всех папках).

2.

Нажать кнопку Новая.

3.

Только для группы пользователей «Эксперты»:

открывается окно Выбор маски. Выделить нужную маску и нажать кнопку OK для подтверждения.

4.

Ввести имя новой программы и нажать кнопку OK для подтверждения.

В группе «Пользователи» нельзя выбрать маску. По умолчанию создается программа типа «Модуль».
