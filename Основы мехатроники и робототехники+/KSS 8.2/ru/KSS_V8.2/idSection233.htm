8.3.4 Программирование блока Spline

Описание

С помощью блока Spline можно объединить в единое движение несколько сегментов SPL, SLIN и/или SCIRC. Блок Spline, не содержащий сегментов, не является командой движения.

Spline-блок может содержать:

■

сегменты Spline (количество ограничено только емкостью памяти);

■

PATH-триггер;

■

комментарии и пустые строки;

■

встроенные команды из технологических пакетов, управляющие функциями Spline.

Блок Spline не может содержать иных команд, например, команд распределения переменных или логических команд. Блок Spline не может инициировать останов предварительной процедуры.

Условие

■

Программа выбрана.

■

Выбран режим работы T1.

Порядок действий

1.

Установить курсор в строку, после которой должен быть вставлен блок Spline.

2.

Выбрать последовательность меню Команды > Движение > Блок SРLINЕ.

3.

Во встроенном формуляре установить параметры.

(>>> 8.3.4.1 "Встроенный формуляр блока Spline")

4.

Нажать кнопку Команда OK.

5.

Нажать Формуляр откр./закр. Теперь в блок Spline можно вставить сегменты Spline и другие строки.
