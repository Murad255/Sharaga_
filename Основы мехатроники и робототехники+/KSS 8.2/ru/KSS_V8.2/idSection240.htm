8.3.4.7 Окно опций «Фреймы» (сегмент Spline)

Рис. 8-17: Окно опций «Фреймы» (сегмент Spline)

Поз.

Описание

1

■

True: для этого движения система управления роботом определяет осевые моменты. Это необходимо для распознавания столкновения.

■

False: для этого движения система управления роботом не определяет осевые моменты. Поэтому распознавание столкновения для этого движения невозможно.
