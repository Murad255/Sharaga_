8.3.4.1 Встроенный формуляр блока Spline

Рис. 8-13: Встроенный формуляр блока Spline

Поз.

Описание

1

Имя блока Spline. Система автоматически задает имя. Имя может быть перезаписано.

(>>> 8.1 "Имена во встроенных формулярах")

Для редактирования данных движения коснуться стрелки. Откроется соответствующее окно опций.

(>>> 8.3.4.2 "Окно опций «Фреймы» (блок Spline)")

2

■

CONT: сглаживание целевой точки.

■

[пусто]: точный подвод в целевую точку.

3

Скорость действительна по умолчанию для всего блока Spline. Возможно дополнительно определить скорость для отдельных сегментов по отдельности.

■

0,001 – 2 м/с

4

Имя записи данных движения. Система автоматически задает имя. Имя может быть перезаписано.

Для редактирования данных движения коснуться стрелки. Откроется соответствующее окно опций.

(>>> 8.3.4.3 "Окно опций «Параметры движения» (блок Spline)")

Данные движения действительны по умолчанию для всего блока Spline. Возможно дополнительно определить их для отдельных сегментов по отдельности.
