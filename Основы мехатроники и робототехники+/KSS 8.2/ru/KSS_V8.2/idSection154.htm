5.8.5 Проверка данных нагрузки онлайн

Конфигурация

Конфигурация OLDC возможна в следующих случаях:

■

При ручном вводе данных инструмента

(>>> 5.7.1.5 "Цифровой ввод")

■

При отдельном вводе данных нагрузки

(>>> 5.8.3 "Ввод данных полезной нагрузки")

В том же окне, в котором вводятся данные нагрузки, отображаются следующие поля:

Рис. 5-35: Проверка данных нагрузки онлайн

Поз.

Описание

1

TRUE: OLDC для отображенного в этом же окне инструмента активно. При повышенной или недостаточной нагрузке выполняются определенные реакции.

FALSE: OLDC для отображенного в этом же окне инструмента неактивно. При повышенной или недостаточной нагрузке реакция отсутствует.

2

Здесь можно определить, какая реакция должна происходить при повышенной нагрузке.

■

Нет: без какой-либо реакции.

■

Предупреждение: система управления роботом выдает следующее сообщение о состоянии: При проверке нагрузки робота (инструмент {№}) определена перегрузка.

■

Останов робота: система управления роботом выдает квитируемое сообщение с таким же содержанием, как при предупреждении. Робот останавливается по схеме останова STOP 2.

3

Здесь можно определить, какая реакция должна происходить при недостаточной нагрузке. Возможные реакции аналогичны вариантам при повышенной нагрузке.

Реакции можно изменить в программе KRL посредством системной переменной $LDC_CONFIG.
