9.1 Сообщения об ошибках, внешняя автоматика

№

Текстовое сообщение

Причина

P00:1

PGNO_TYPE неправильное значение
допустимые значения (1,2,3)

Указан неправильный тип файла номера программы.

P00:2

PGNO_LENGTH неправильное значение
диапазон значений 1 ≤ PGNO_LENGTH ≤ 16

Неправильно определена ширина бита номера программы.

P00:3

PGNO_LENGTH неправильное значение
допустимые значения (4,8,12,16)

Если для чтения номера программы выбран формат BCD, необходимо установить подходящую ширину бита.

P00:4

PGNO_FBIT неправильное значение
находится вне диапазона $IN

Для первого бита номера программы указано значение "0" или отсутствующий вход.

P00:7

PGNO_REQ неправильное значение
находится вне диапазона $OUT

Для выхода, через который запрашивается номер программы, указано значение "0", или указан отсутствующий выход.

P00:10

Ошибка передачи
неправильная четность

При проверке четности обнаружилось несоответствие. Вероятно, произошла ошибка при передаче.

P00:11

Ошибка передачи
неправильный номер программы

Система управления верхнего уровня передала номер программы, для которого в файле CELL.SRC отсутствует ветвь CASE.

P00:12

Ошибка передачи
неправильный код BCD

Попытка ввести номер программы в формате BCD привела к недействительному результату.

P00:13

Неправильный режим работы

Интерфейс ввода/вывода не был активирован, т. е. системная переменная $I_O_ACTCONF имеет в данный момент значение FALSE. Это могло иметь следующие причины:

■

Переключатель режимов работы не в положении "Внешняя автоматика".

■

Сигнал $I_O_ACT имеет в данный момент значение FALSE.

P00:14

Подвод к позиции Home в режиме работы Т1

Робот не достиг позиции HOME.

P00:15

Неправильный номер программы

При "1 из n" задано больше одного входа.
