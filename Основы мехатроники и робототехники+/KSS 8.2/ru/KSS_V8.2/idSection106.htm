5.5.1 Методы юстировки

Возможно применимые для робота методы юстировки зависят от того, каким типом измерительного патрона он оснащен. Типы различаются визуально по размеру своих защитных крышек.

Юстировка по исходным данным

Юстировка по исходным данным для роботов с измерительным патроном для EMD используется только для юстировки после определенных ремонтных процедур.
