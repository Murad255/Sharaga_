4.15.11 Показ/обработка данных робота

Условие

■

Выбран режим работы T1 или T2.

■

Ни одна из программ не выбрана.

Порядок действий

■

В главном меню выбрать пункт Ввод в эксплуатацию > Данные о роботе.

Описание

Рис. 4-33: Окно Данные о роботе

Поз.

Описание

1

Серийный номер

2

Время работы. Счетчик времени работы робота действует, если приводы включены. Альтернативная возможность отображения времени работы робота – через переменную $ROBRUNTIME.

3

Имя параметров машины

4

Имя робота. Имя робота можно изменить.

5

Данные системы управления роботом можно заархивировать по сетевому пути.

(>>> 6.8.3 "Архивирование данных в сети")

Здесь указывается путь, по которому данные архивируются.

6

Если для архивирования по данному сетевому пути необходимо имя пользователя и пароль, их можно ввести здесь же. В этом случае их не надо будет вводить каждый раз при архивировании.

7

8

Это поле показывается только в том случае, если флажок Включать имена роботов в имена архивов. не активирован.

Здесь можно ввести имя файла архива.

9

■

Флажок установлен: в качестве имени файла архива используется имя робота. Если имя робота не установлено, используется имя archive.

■

Флажок не установлен: для файла архива можно ввести другое имя.

Эти кнопки недоступны в группе «Пользователи».
