В таблице сгибов можно указывать: Допуск сгиба, Уменьшение сгиба, Коэффициент K. 

Таблицы сгибов по варианту 1 определяют длину сгиба на основании заданных значений соответствующих параметров для диапазона углов сгиба и радиусов сгиба для каждой используемой толщины материала.
При выборе промежуточных значений углов сгиба, радиусов сгиба или толщин материалов длина сгиба аппроксимируется на основе данных таблиц. Использование параметров выходящих за приделы определенных в таблицах не допускается.

Таблицы сгибов (с указанными допуском сгибов и уменьшениями сгибов) по варианту 2 определяют длину сгиба на основании заданных значений соответствующих параметров для диапазона толщин материала и радиусов сгиба, указанных для угла сгиба 90 град. При использование отличных от 90 град углов сгиба, длина сгиба определяется прямо пропорционально углу сгиба (например, при угле сгиба 45 град, заданный или вычисленный по уменьшению сгиба для угла сгиба 90 град допуск сгиба уменьшается вдвое). Использование параметров выходящих за приделы определенных в таблице не допускается.

Таблица сгибов с указанным коэффициентом K по варианту 2 определяет длину сгиба по коэффициентам заданным для диапазона углов сгиба и отношений Радиус сгиба / Толщина материала. Использование параметров выходящих за приделы определенных в таблице не допускается.
